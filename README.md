# 🏠 Project RedForge - Home Cybersecurity Lab

> "Building a professional cybersecurity home lab from the ground up."

---

# 📖 Overview

This repository documents the design, implementation, and continuous development of my personal cybersecurity home lab.

The purpose of this lab is to gain practical, hands-on experience with enterprise technologies, networking, Windows administration, Linux, Active Directory, SIEM, scripting, cloud security, and offensive security.

Everything in this repository represents my learning journey as I transition from 13 years of military leadership into cybersecurity.

---

# 🎯 Objectives

- Build a professional enterprise-style home lab
- Learn networking fundamentals
- Master Windows administration
- Learn Linux administration
- Build an Active Directory domain
- Deploy SIEM solutions
- Develop Python automation
- Practice offensive security in a safe environment
- Document everything
